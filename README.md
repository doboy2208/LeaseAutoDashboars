# Levend Lease Dashboard Webapp v2

Verbeteringen:
- Navigatiebar met views: Dashboard, Auto's, Scoremodel, Functiematrix en Ranglijst.
- Betere responsive layout voor mobiel/tablet/desktop.
- Data gescheiden in `data/cars.js` en `data/scoring.js`.
- PWA-ready voor GitHub Pages en iPhone beginscherm.

Publiceren:
1. Upload alles naar de root van je GitHub repository.
2. Settings → Pages → Deploy from branch → main / root.
3. Open de Pages-url op iPhone → Deelknop → Zet op beginscherm.
