# Levend Lease Dashboard PWA

Dit pakket is klaar voor GitHub Pages en kan als PWA op je iPhone-beginscherm worden gezet.

## Lokaal testen
Op desktop: open `index.html`.

## Publiceren via GitHub Pages
1. Maak een nieuwe GitHub repository, bijvoorbeeld `lease-dashboard`.
2. Upload alle bestanden uit deze map naar de repository-root.
3. Ga naar Settings → Pages.
4. Kies Source: `Deploy from a branch`.
5. Kies Branch: `main` en folder `/root`.
6. Open de gegenereerde GitHub Pages URL op je iPhone.
7. Safari → Deelknop → Zet op beginscherm.

## Nieuwe auto toevoegen
Voeg een nieuw object toe aan de `CARS` array in `cars.js`.
